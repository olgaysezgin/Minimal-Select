Standard Select:

Minimal Select:

Usage: 

                    
$('.selector').minimal_select();

Options:

  • placeholder attr, default placeholder text on standard select box placeholder attribute.
                    
placeholder="Select one"
  • list_top, top position for drop down.
  • lit_max_height, max height for drop down.
  • select_width, ui select width.
  • arrow_padding, ui select right arrow width.
  • placeholder_text, default placeholder text.
                    
$('.selector').minimal_select({
  list_top: 34,
  lit_max_height: 156,
  select_width: 0,
  arrow_padding: 44,
  placeholder_text: ""
});
